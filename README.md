# Kathys-baKery

#### An Bakery Application that will allow users to shop for baked goods and also allow management to maintain inventory by updating, removing or adding a new item to the bakery.

## Technology Used

- ### HTML

- ### Cascading Style Sheet(CSS)

- ### Javascript

- ### Google Fonts

- ### Express

- ### NodeJS

- ### MONGODB

### Mongoose
